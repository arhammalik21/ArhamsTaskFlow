# Arham’s TaskFlow

Arham’s TaskFlow is a simple and intuitive To-Do List web application built with Python and Flask.  
It helps users organize their tasks by allowing easy adding and deleting of tasks in a clean interface.

## Features

- Add new tasks quickly  
- Delete tasks effortlessly  
- Minimalistic and user-friendly design

## Technologies Used

- Python  
- Flask web framework  
- HTML & Jinja2 templating
